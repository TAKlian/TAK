Core 02
  • Fairness, transparency, openness and autonomy
    With the introduction of blockchain technology and ideas, all institutions or individuals can fairly participate in the collection, sharing and open movement of IOT data without any endorsement of central institutions. All institutions or individuals can join the open data platform of Internet of all things and become a node of Tak data link just by identifying these rules based on cryptography and data. No one can control the opening, trading or use of the data platform.
  • Data can be identified and traceable
    Through the blockchain's distributed ledger, all data upload, update, transaction or use behaviors can be recorded safely and transparently. Once confirmed, these behavior records cannot be tampered with, which makes all data on the open platform have the function of confirming rights and traceability.
  • Secure trusted data transactions
    The point-to-point data transaction fundamentally avoids the possibility of data precipitation on the platform. Using smart contract, data transparent open pricing, to ensure the fairness of transactions. The purchased data is encrypted and transmitted by the data provider with the public key of the data purchaser, and only the private key of the purchaser can be decrypted, so as to avoid data leakage. The data on the platform has a unique signature. If the data buyer resells the data, the revenue will be recorded in the initial data provider. Data scoring mechanism based on smart contract and community consensus.
  • Multidimensional big data integration
    The massive vertical applications share data based on consensus mechanism, forming an open platform for big data sharing, so as to achieve a more complete and accurate multi-dimensional data portrait.
  • User privacy protection and user data confirmation
    Use unified digital identity to map different identities of users in multiple devices and scenarios on the network. It can be used for cross screen and cross application combination of user data and unified login. The user's personal data is completely owned by the user, and the user fully grasps the private key, and the user decides to whom to open the authorization and how to charge. The whole process can be tracked.
